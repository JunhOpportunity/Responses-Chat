# RE-CHAT

## Outline
##### 이 프로젝트는 저에 대한 주변 사람들의 반응, 평판 등을 작성할 수 있는 프로젝트입니다.
##### 현재 한 명, 즉 저에 대한 평판만 작성이 가능하지만, 약간의 기능을 더 추가하면 다양한 사람들의 평판을 작성할 수 있도록 만들 수 있습니다.
